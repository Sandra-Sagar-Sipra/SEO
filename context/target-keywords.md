## Topic Cluster Structure

Each cluster has:
- **Pillar Keyword**: Main, higher-volume keyword (competitive — target with cornerstone content)
- **Cluster Keywords**: 5–10 related subtopic keywords
- **Long-Tail Keywords**: Specific, lower-competition phrases with strong commercial intent
- **Search Intent**: Informational (learn), Commercial (compare/evaluate), Transactional (ready to buy)

## Usage Guidelines

### When Writing New Content
1. Check which cluster the topic belongs to
2. Target the appropriate cluster or long-tail keyword
3. Reference the pillar service page in that cluster
4. Include LSI keywords naturally throughout
5. Link to related cluster articles

### When Planning Content Calendar
1. Prioritise TOFU educational content (builds topical authority fast)
2. Build commercial intent content for each service area
3. Publish one industry-specific piece per month
4. Target "why AI fails" angle — directly supports Aiquire's messaging
